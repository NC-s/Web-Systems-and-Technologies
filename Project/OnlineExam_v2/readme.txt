Setup:
1. Open XAMPP, start the "Apache Web Server" and "MySQL Database".
2. Put the "onlineExam_v2" folder under XAMPP "htdocs" folder.
3. Create a empty database call "online_examination" in XAMPP.
4. Import the file "online_examination.sql" to the "online_examination" database.
5. Open Google Chrome, type the url as "localhost/onlineExam_v2/index.php".
