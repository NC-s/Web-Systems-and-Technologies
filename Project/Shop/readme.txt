Setup:
1. Open XAMPP, start the "Apache Web Server" and "MySQL Database".
2. Put the "OnlineStore" folder under XAMPP "htdocs" folder.
3. Create a empty database call "onlineshop" in XAMPP.
4. Import the file "onlineshop.sql" to the "onlineshop" database.
5. Open Google Chrome, type the url as "localhost/OnlineStore/MainPage.php".
6. Use the following test account for demonstration purposes. Create a new account is also welcome.
